## 项目说明

本项目为济南护理职业学院教学成果奖申报展示网站，全面呈现"全人健康·全链协同·处方育人"健康教育体系的创新理念、实践成果与荣誉奖励。

## 技术栈

- Vue 3 + Vite
- SVG SMIL 动画
- 响应式布局

## 本地开发

```bash
npm install
npm run dev
```
